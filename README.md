<h4 align="center">CortexLink is a search engine for searching about the topic you want.</h4>
CortexLink is a search engine for searching about the topic you want, the return answer of this tool is equal to: educational links, Exploit links and vulnerability PoCs and even CVEs of a product you are searching about, etc.

## Install CortexLink

CortexLink requires **python** to install & run successfully. Run the following command to install the latest version -

<details>
  <summary>Windows</summary>
  
  ```sh
  git clone github.com/Symbolexe/CortexLink
  pip install Requirements.txt
  ```
  ⚠️ **You can download .exe file in Releases for windows** ⚠️

</details>
<details>
  <summary>Linux</summary>

### Usage
```sh
python CortexLink.py
```

Here are all the switches it supports.
```console
Usage :
    Commands :
        exit      - Exit The Program
        me        - show your subscription & your system information
    Search Options :
        cve       - Go To Search about CVE Links
        exploit   - Go To Search about Exploit Links
        poc       - Go To Search about PoC Links
        learn     - Go To Search about Education & Learning Links
```

### Running CortexLink
```sh
python CortexLink.py
```

Now you just need to type your word.
